# TeamFinder — Платформа для поиска команды

**Автор проекта:** Мухамадеев Артём Маратович

Добро пожаловать в **TeamFinder** — современную платформу для создания проектов и поиска единомышленников!

## 🚀 Возможности проекта

### Для всех пользователей
- **Просмотр проектов**: Изучайте актуальные проекты с удобной пагинацией (12 проектов на странице)
- **Профили участников**: Знакомьтесь с авторами проектов и их навыками
- **Поиск по навыкам**: Фильтруйте пользователей по интересующим технологиям

### Для зарегистрированных пользователей
- **Создание проектов**: Публикуйте свои идеи и находите команду
- **Участие в проектах**: Присоединяйтесь к интересным проектам других авторов
- **Избранное**: Сохраняйте понравившиеся проекты в закладки
- **Управление профилем**: Редактируйте личную информацию, загружайте аватар, добавляйте навыки
- **Навыки**: Указывайте свои компетенции с автодополнением

### Для владельцев проектов
- **Редактирование**: Обновляйте описание и статус проекта
- **Управление участниками**: Видите список участников вашего проекта
- **Завершение проекта**: Меняйте статус на "закрыт" при необходимости

## 🛠 Технологии

- **Backend**: Django 4.x
- **База данных**: PostgreSQL 16
- **Контейнеризация**: Docker & Docker Compose
- **Frontend**: HTML5, CSS3 (современный градиентный дизайн), JavaScript

## 📦 Установка и запуск

### Вариант 1: С использованием Docker Compose (рекомендуется)

1. **Клонируйте репозиторий:**
   ```bash
   git clone <url-репозитория>
   cd teamfinder
   ```

2. **Создайте файл `.env` в корне проекта:**
   ```env
   SECRET_KEY=your-secret-key-here
   DEBUG=True
   ALLOWED_HOSTS=localhost,127.0.0.1
   
   POSTGRES_DB=teamfinder
   POSTGRES_USER=postgres
   POSTGRES_PASSWORD=postgres
   DB_HOST=db
   DB_PORT=5432
   ```

3. **Запустите проект:**
   ```bash
   docker-compose up --build
   ```

4. **Откройте браузер и перейдите по адресу:**
   ```
   http://localhost:8000
   ```

### Вариант 2: Локальная установка

1. **Создайте виртуальное окружение и активируйте его:**
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux/Mac
   source venv/bin/activate
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройте переменные окружения** (создайте `.env` файл как показано выше)

4. **Выполните миграции:**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

5. **Создайте суперпользователя (опционально):**
   ```bash
   python manage.py createsuperuser
   ```

6. **Запустите сервер разработки:**
   ```bash
   python manage.py runserver
   ```

7. **Откройте браузер:** `http://127.0.0.1:8000`

## 📁 Структура проекта

```
teamfinder/
├── config/              # Настройки Django проекта
│   ├── settings.py      # Основные настройки
│   └── urls.py          # Корневой URLconf
├── users/               # Приложение пользователей
│   ├── models.py        # Модель User и Skill
│   ├── views.py         # Представления для работы с пользователями
│   └── forms.py         # Формы регистрации, входа, редактирования
├── projects/            # Приложение проектов
│   ├── models.py        # Модель Project
│   ├── views.py         # CRUD операции для проектов
│   └── forms.py         # Формы создания/редактирования проектов
├── templates/           # HTML шаблоны
├── static/              # Статические файлы (CSS, JS, изображения)
├── media/               # Загружаемые пользователем файлы
└── docker-compose.yml   # Конфигурация Docker Compose
```
